## MOOD ANALYSIS AND INSIGHTS DASHBOARD
## GOAL-
TO ANALYZE THE MOOD SENTIMENT OF DIFFERENT CUSTOMERS AND ANALYSIS FOR BETTER SALES

## APPROACH - 
- DATA CLEANING USING PANDAS
- CONNECTED PYTHON WITH SQL DATABASE
- MOOD SCORE USING TEXTBLOB
- VISUALIZATION USING PLOTLY
- AUTOMATED FOR EXTRACTION AND PROCESSING

## TOOLS-
- PYTHON
- PLOTLY
- MYSQL
- TEXTBLOB

## Key Insights

- Users aged between 30–40 tend to leave more reviews compared to other age groups.
- Gen Z and Millennials are the most active users, contributing a higher number of reviews, especially positive ones.
- The "Tops" category receives the highest number of both positive and negative reviews, indicating high engagement and popularity.
- The "Trends" department has the lowest number of positive reviews, suggesting lower customer interest or engagement.
- Overall sentiment is positive, with more positive reviews than negative, which is a good sign for the business.

## Business Recommendations

- Focus on targeting younger audiences (18–26 age group), particularly Gen Z, as they are highly active and influential.
- Introduce more Gen Z–oriented and trendy clothing to increase engagement and sales.
- Improve "Trends" department, as it currently underperforms.
- Optimize product offerings in the "TREND" category by analyzing both positive and negative feedback to improve customer satisfaction.
- use targetted social media strategies to engage genz and increse customer engagement.
